# 🌐 Personal Portfolio – Emilia Zabrzanska

This repository contains the source code for my personal portfolio website.  
It showcases my academic background, professional experience, and projects in **data science, astrophysics, software development, and training**.  

---

## 📘 Overview

The portfolio is designed to highlight my journey from a **Maths and Physics graduate** through to my current **MPhil in Data Intensive Science at the University of Cambridge**, and my aspiration to continue on to a **PhD integrating AI into lifesaving and lifeguarding technology**.  

The site also features my professional work in **operations, training, and leadership**, alongside technical projects such as:  
- **NPLQ Revision App** – a full-stack React/Firebase web app  
- **Exoplanet Transit Photometry** – Python + HOPS pipeline analysis of XO-2Nb  
- **BAO Cosmology Project** – investigating large-scale structure in galaxy surveys  
- **Swing Safety Modelling** – applied mathematics for real-world safety standards  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Styling:** Custom CSS (responsive layouts, consistent design system)  
- **Deployment:** Vercel / GitHub Pages  

---

## 🚀 Features

- **About Me** – academic background, professional experience, and future aspirations  
- **Projects** – detailed write-ups of coding, physics, and applied maths projects  
- **Blog** – reflections on development and learning (e.g. building my NPLQ app)  
- **Contact** – links for email, LinkedIn, and CV
